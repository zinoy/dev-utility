Spritesheet Serializer for CreateJS
Paste content or drop file generated for CreateJS . Note: keys in animations property must ended with numbers (eg. "ani0001").
Include animate.js file in your project, and initialize object using genAni(jsonObj); when JSON file has been loaded. Then, you may create Sprite instance with var sprite = new ani.{{ sprite.inputClass || 'className'}}();. Class name must start with letter and at least 2 characters length. Only letters and numbers allowed.
